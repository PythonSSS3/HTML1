HTML

HTML describes the structure of a web page. HTML elements are the building blocks of HTML pages. HTML tags are written inside angle brackets (<>). HTML code is written in Notepad, Wordpad, or any other editor and is viewed on web browsers like Internet Explorer, Google Chrome, etc.

Basic HTML TAGS

There are some basic HTML tags:

  • html tag: All the tags are enclosed within this tag. This tag represents the document as an HTML document.
  • head tag: This tag describes the header of an HTML page.
  • title tag: This tag defines the title for the HTML page.
  • body: This tag includes the content of your HTML Page.
  • br: This tag breaks the text to the next line.
  • h1 to h6: HTML provides 6 levels of heading: h1 to h6. These levels are used to define headings and subheadings in a web page. Browsers add one line before and after each heading.

Other HTML tags

HTML provides tags that enable you to add images, hyperlinks, lists, tables, frames, and other items in web pages. These items make web pages more efficient, attractive, and readable.

Image tag

You can insert images in web pages to make them more beautiful and improve the design and appearance. In an HTML document, images are defined with the img tag. The image tag is an empty tag.

Attribute Description
src To specify the URL (web address) of the image.
alt To provide alternative text for an image, if for any reason the user cannot view the image.

Hyperlink

A hyperlink is generally an underlined text or an image. A hyperlink helps you to jump to the linked web page by clicking on it. This hyperlink text is generally seen in blue colour. When the mouse is hovered over it, the mouse pointer changes to a pointed finger and the linked text colour may also get changed.

The anchor tag is used for creating hyperlinks. It is a container tag. The attribute of the anchor tag is:

Attribute Description
href To specify the URL (web address) of the web page which is to be linked.

Lists: ordered list, unordered list, description list

You can set the content of the web pages in the form of lists by creating any of these types of lists: Ordered list, Unordered list, and Definition or description list:

  1. Ordered list: An ordered list contains items in a numbered manner. It is defined using the ol tag. List items are marked with numbers by default. The attributes of the ordered list tag are:
    Attribute Description type To set the style of numbering. Type attribute has different options for numbering:
    • "1" defines the items with numbers (by default).
    • "A" defines the items with uppercase letters.
    • "a" defines the items with lowercase letters.
    • "I" defines the items with uppercase Roman numbers.
    • "i" defines the items with lowercase Roman numbers.
    start To define the starting number for the type attribute.
  2. Unordered list: An unordered list contains items that can be listed without any order, i.e., in the form of bullets or other symbols. The unordered list is a container tag. The attribute of the unordered list is:
    Attribute Description
    type To set the style of bullets. Type attribute has different options:
    • "circle" defines a hollow bullet.
    • "disc" defines a solid round bullet.
    • "square" defines a solid square bullet.
  3. Description list: A description list contains items and their description. The description list tag is a container tag. Description list consists of two parts:
    • dt tag defines the item name.
    • dd tag defines the description of the item.

Tables

Tables are used to present data in tabular format. A table is defined using the table tag. It is a container tag.

Different kinds of tables can be made using the following tags with the table tag:

  • caption: This tag is used for defining the title of the table.
  • th: This tag is used for defining row and column headings.
  • tr: This tag is used for defining table rows.
  • td: This tag is used for defining table data, that is, the data of the rows.

The attributes of the table tag are:

Attribute Description
border To form a border around the table.
bordercolor To specify the color of the table's border.
cellpadding To add more space between the cell border and the cell contents.
cellspacing To put more space around each cell, i.e., the space between cells.
width To set the width of the table.
height To set the height of the table.
align To align the table left, right, or center.
